# 前端组件

------

| 模块分类 | 组件文件                                       | 功能概述                           | 典型使用场景           |
| -------- | ---------------------------------------------- | ---------------------------------- | ---------------------- |
| 导航模块 | [`nav.html`](./nav.html)                       | 网站主导航栏，支持响应式与下拉菜单 | 页面顶部全局导航       |
|          | [`nav-search.html`](./nav-search.html)         | 导航栏内嵌搜索框                   | 与 `nav.html` 组合使用 |
| 文章模块 | [`article-detail.html`](./article-detail.html) | 渲染markdown文章                   | 文章详情页             |
|          | [`article-search.html`](./article-search.html) | 搜索结果列表展示                   | 博客页的搜索框         |
|          | [`article-toc.html`](./article-toc.html)       | 自动生成文章目录（TOC）            | 详情页右侧浮动目录     |
|          | [`blog-article.html`](./blog-article.html)     | 文章卡片，展示标题、日期等         | 博客页显示多篇博客     |
| 四季图   | [`seasons.html`](./seasons.html)               | 根据季节展示主题装饰动画           | 首页背景               |
| 公共模块 | [`footer.html`](./footer.html)                 | 页脚，展示版权、备案号、社交链接   | 页面底部               |
|          | [`backToTop.html`](./backToTop.html)           | 回到页面顶部                       | 每个页面的右下角       |

## 组件详细说明

### `nav.html` — 网站导航栏
- **功能**：展示博客主导航（首页、分类、归档、关于等）
- **特点**：支持响应式布局与下拉菜单
- **依赖**：可与 `nav-search.html` 组合使用

---

### `nav-search.html` — 导航搜索框
- **功能**：在导航栏中提供文章搜索入口
- **用途**：快速跳转到目标文章

---

### `article-detail.html` — 文章详情页模板
- **功能**：渲染<u>单篇</u>文章内容，包括标题、作者、日期等信息
- **建议**：与 `article-toc.html` 搭配使用，支持目录导航

---

### `article-search.html` — 搜索结果页
- **功能**：博客页的搜索框
- **特点**：可高亮搜索关键词、支持分页扩展

---

### `article-toc.html` — 文章目录（Table of Contents）
- **功能**：生成文章目录索引，点击跳转至对应段落
- **常见用法**：详情页右侧固定导航

---

### `blog-article.html` — 文章卡片组件
- **功能**：卡片，在博客页展示多篇文章简要信息（标题、摘要、日期）
- **适用场景**：博客首页、分类页、归档页

---

### `seasons.html` — 季节主题组件
- **功能**：根据季节动态展示不同背景特效
- **目的**：增强视觉体验与氛围感

---

### `footer.html` — 页脚组件
- **功能**：展示版权、备案号及社交媒体链接
- **当前状态**：为空文件，可后续补充内容

### `backToTop.html` — 回到顶部组件

- **功能**：回到页面顶部
- **位置**：每个页面的右下角

需设置两部分：

- 锚点

	> 在页面的顶部设置锚点
	>
	> ```html
	> <div id="top-anchor"></div>
	> ```
	>
	> 通过锚点回到顶部
	>
	> ```html
	> <a href="#top-anchor"></a>
	> ```

- 挂载点

	> ```html
	> <!-- 回到页面顶部组件的挂载点 -->
	> <div class="backToTop__container"></div>
	> ```
